# Quick Start Guide

## Prerequisites
- Python 3.10+
- Virtual Environment

## Installation

1. **Navigate to project:**
```bash
cd 'C:\Users\NAZIM\OneDrive\Desktop\AI Bot'
```

2. **First time only - Index documents:**
```bash
.\venv\Scripts\python.exe ingest.py
```

## Usage

### Ask a Single Question
```bash
.\venv\Scripts\python.exe ask_bot.py "Your question here"
```

### Examples:
```bash
.\venv\Scripts\python.exe ask_bot.py "What is Python?"
.\venv\Scripts\python.exe ask_bot.py "Explain machine learning"
.\venv\Scripts\python.exe ask_bot.py "What are neural networks?"
```

### Interactive Mode
```bash
.\venv\Scripts\python.exe ask_bot.py
```
Then type your questions and press Enter. Type `quit` to exit.
